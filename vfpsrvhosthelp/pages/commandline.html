Distribution

VFPSrvHost Package

The distribution contains the following directories:

vfpsrvhostContains the vfpsrvhost.exe and all supporting files.
vfpexampleserviceContains an example service project.

Deployment

The vfpsrvhost.exe executable has only one dependency which is XMLLite - this system library is part of any Windows OS since XP SP3 / Windows Server 2003. You can use simple xcopy deployment for the executable itself, the VFP COM classes which implement the service have to be registered on the system or you have to create a manifest file (vfpsrvhost.exe.manifest) for regfree deployment.
