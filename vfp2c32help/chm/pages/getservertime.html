GetServerTime

The retrieves the current time from a windows server.

GetServerTime(cServer [, nTimeZone])

Parameters

cServer

The DNS or NetBIOS name of the remote server from which to retrieve the time.

Note

Windows NT: This string must begin with \\.

nTimeZone (optional)

default = 1

The timezone into which the returned time should be converted.

One of the following values:

TimezoneMeaning
1UTC
2The timezone of the local computer.
3The timezone of the server.

Return Value

A datetime value.
