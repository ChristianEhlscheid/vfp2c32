AbortRasConnectionNotification

Aborts the thread monitoring the RAS connection started by RasConnectionNotificationEx.

AbortRasConnectionNotification(nThreadHandle)

Parameters

nThreadHandle

The thread handle returned from RasConnectionNotificationEx.

Return Value

.T. if the thread monitoring the RAS connection was terminated, .F. otherwise.
