ADesktopArea

Stores the dimension of the visible desktop (not including the system taskbar or any application desktop toolbars) area into an array.

ADesktopArea(cArrayName)

Parameters

cArrayName
On return the array contains the following values from the RECT structure.
ElementContent
1Left
2Right
3Top
4Bottom

Return Value

Always .T.
