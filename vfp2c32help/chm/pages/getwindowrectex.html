GetWindowRectEx

Retrieves the dimensions of the bounding rectangle of the specified window.

GetWindowRectEx(nHwnd, cArrayName)

Parameters

nHwnd

A handle to a window.

cArrayName
On return the array contains the following values from the RECT structure.
ElementContent
1Left
2Right
3Top
4Bottom

Return Value

Always .T.
