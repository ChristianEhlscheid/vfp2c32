VFP2CSys

Provides access to library internal resources or changes global library behaviour.

VFP2CSys(nOption [, nNewValue])

Parameters

nOption
OptionReturn value
1HMODULE of vfp2c32.fll
2HANDLE to library internal heap used for marshaling
3default codepage used for Ansi - Unicode conversion - defaults to CP_ACP
4automatic Unicode to Ansi conversion in SqlExecEx - defaults to .T.
nNewValue (optional)

The following options can be altered.

OptionDescription
3nNewValue should be a new codepage value which is used in all Ansi to Unicode string conversions
4nNewValue should be .T. or .F.

Return Value

The return value depends on the passed in value.
