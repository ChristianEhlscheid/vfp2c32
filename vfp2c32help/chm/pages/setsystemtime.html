SetSystemTime

Sets the current system time and date.

SetSystemTime(tTime [, bUTCTime])

Parameters

tTime

A datetime value.

bUTCTime (optional)

default = .F.

If .T. the passed in datetime is assumed to be in UTC time.

Return Value

Always .T.
