RasGetConnectStatusEx

Retrieves information on the current status of the specified remote access connection into an array.

RasGetConnectStatusEx(nRasConn, cArrayname)

Parameters

nRasConn

A valid HRASCONN handle for which to retrieve the status information.

cArrayname
On return the array contains the following information from the RASCONNSTATUS structure.
ElementContent
1RASCONNSTATE
2Errorcode
3Devicetype
4Devicename
5Phonenumber

Return Value

Always .T.
