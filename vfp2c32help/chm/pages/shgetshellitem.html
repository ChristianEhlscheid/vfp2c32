SHGetShellItem

Returns a scriptable proxy object over the IShellItem2 interface.

SHGetShellItem(cFilePath)

Parameters

cFilePath

A path to a filename.

Return Value

Object

Remarks

The returned object supports the following methods:
-GetProperty(cPropertyName) - retrieves the specified property
-GetPropertyDescriptionList(cPropGroup (optional)) - returns an object to enumerate available properties (IPropertyDescriptionList)
-GetPropertyStore() - returns the property store (IPropertyStore) to read/write properties

IPropertyDescriptionList supports the following methods:
GetAt(lnIndex)- return the property description (IPropertyDescription) at the passed index (zero based)
GetCount() - return the number of property descriptions
The object also supports an enumeration interface - you can code a
FOR EACH loObj IN loPropertyDescriptionList loop to iterate over all properties.

IPropertyDescription supports the following methods:
GetCanonicalName() - return the fully classified name of the property
GetPropertyType() - returns the underlying type of the property
GetDisplayName() - returns the localized display name

IPropertyStore supports the following methods:
-GetValue(lcProperty) - reads the specified property
-SetValue(lcProperty, vValue) - writes a new value to the specified property
-Commit() - commits the changes to the file

If reading/writing multiple properties it's more efficient to to use the IPropertyStore object,
the normal GetProperty() function on the IShellItem2 will open and then close the file
for each call.
