DT2Timet

Converts a datetime value into a Time_t (Unix) timestamp.

DT2Timet([tTime [, bLocalDateTime]])

Parameters

tTime (optional)

A datetime value.

bLocalDateTime (optional)

default = .F.

If the passed DateTime value is in the local timezone pass .T., if the DateTime value is in UTC pass .F. or omit the parameter.

Return Value

The Time_t (UNIX) timestamp - a numeric value indicating the number of elapsed seconds since 1970/01/01.

Remarks

If you don't pass any parameters to the function it returns the UNIX timestamp of the current time.

Example

? "Current UNIX timestamp", DT2Timet(), DT2Timet(DATETIME(), .T.)
? "Current DateTime in local timezone", DATETIME(), TIMET2DT(DT2TIMET())
? "Current DateTime in UTC", GetSystemTime(.T.), TIMET2DT(DT2TIMET(), .T.)
