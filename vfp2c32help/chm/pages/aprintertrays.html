APrinterTrays

Stores information about the available paper bins of a printer into an array.

APrinterTrays(cArrayName, cPrinter, cPort)

Parameters

cArrayName
On return the array contains the following information.
ColumnContentDatatype
1Name of the printer tray.C
2Tray ID.N
cPrinter

Name of the printer for which to enumerate the trays.

cPort

Name of the port to which the printer is connected.

Return Value

The number of printer trays.
